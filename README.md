# Install-Asterisk-di-Ubuntu-
Cara menginstal Asterisk di Ubuntu pada Virtual Box

<p>$ sudo si</p>
<p>$ apt-get update</p>
<p>$ apt-get upgrade</p>
<p>$ apt-get install asterisk</p>
<p>$ nano /etc/asterisk/sip.conf</p><hr>
<p>[nomor telepon]</p>
<p>type=friend</p>
<p>context=telpon</p>
<p>username=nama kalian</p>
<p>secret=sesuai kalian</p>
<p>host=dynamic</p>
